LiveCodeLab tests

Functional and perceptual tests of LiveCodeLab are run here. This page uses the Jasmine test framework and js-imagediff.

Note 1: the tests probably won't work on their first run (reason unknown). Please refresh this page once.

Note 2: tests won't work if page is opened as file (i.e. via file://... url). If you need to run this locally, a simple server is provided to obviate this (se 'runTests.sh' script).
