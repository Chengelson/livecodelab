SampleBank

The SampleBank is responsible for holding the filepaths to any audio that needs to be loaded by the browser.

It automatically handles returning the ogg or mp3 file path.

(defined in /docs/deleteme/sourcesWithBlockComments/coffee/sound/samplebank.coffee:8)

  load: (name, path) ->
    soundNumber = @sounds.length
    @sounds.push
      name: name
      path: path + "." + @fileType

    @soundsByName[name] = soundNumber
    soundNumber

  getByName: (name) ->
    @sounds[@soundsByName[name]]

  getByNumber: (number) ->
    @sounds[number]
